    Hexo

    1 Hexo

    Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

    2 安装

    前提:

    1.node.js
    npm（全称Node Package Manager，即node包管理器）是Node.js默认的、以JavaScript编写的软件包管理系统。
    2.git
    创建gh-pages.

    安装：

    npm install in global mode (ie, with -g or –global appended to the command),it installs the current package context (ie, the current working directory) as a global package.

    3 创建工程

    创建文件夹testblog并初始化。

    进入文件夹初始化工程helloworld.此时会在/blog/sources/_post/目录下生成HelloWorld.md文件.

    使用Hexo引擎将Markdown格式的文件解析成可以使用浏览器查看的HTML文件，HTML文件存储在blog/public目录下。

    打开命令行提示的地址，一般是http://0.0.0.0:4000/ ，即可以看到Hexo网站。hexo server -p 5000可以指定端口。如果提示找不到server命令则需要运行命令$ npm install hexo-server –save,Hexo3.0之后把server独立出来了，需要单独安装.

    部署到远端。

    git部署：

    1. 安装hexo-deployer-git

    2. 在 _config.yml 中修改参数

    3. 使用sshpublickey.

      -t 加密，-b 长度， -C comments

    4 配置

    https://hexo.io/plugins/

    5 注意

    • 1.Hexo内置的MD渲染引擎不支持

      /npm install -g webpack
      这样的写法。(怎么直接灰了)
